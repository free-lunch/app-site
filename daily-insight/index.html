Today Card : Daily Insights 개인정보처리방침

시행일: 2026년 3월 29일

ODF(이하 "개발자")는 Today Card : Daily Insights(이하 "앱") 이용자의 개인정보를 소중히 여기며, 관련 법령에 따라 아래와 같이 개인정보처리방침을 수립하여 공개합니다.

1. 수집하는 개인정보

본 앱은 서버를 사용하지 않으며, 사용자의 개인정보를 직접 수집하지 않습니다. 모든 데이터는 사용자의 기기 내에만 저장됩니다.

2. 광고 및 제3자 서비스

본 앱은 다음의 제3자 서비스를 사용합니다:

  • Google AdMob: 광고 제공을 위해 기기 광고 식별자(광고 ID) 등의 정보를 수집할 수 있습니다.
  • Firebase Analytics: 앱 사용 통계를 위해 익명화된 사용 데이터를 수집합니다.
  • Google ML Kit: 번역 기능은 기기 내에서 처리되며, 외부로 데이터가 전송되지 않습니다.

제3자 서비스의 개인정보 처리는 해당 서비스의 정책에 따릅니다:

  • Google 개인정보처리방침

3. 개인정보의 제3자 제공

개발자는 사용자의 개인정보를 제3자에게 직접 제공하지 않습니다. 다만, 위 제3자 서비스를 통해 간접적으로 수집되는 정보는 해당 서비스 제공자의 정책에 따릅니다.

4. 데이터 보관 및 삭제

앱 내 데이터는 사용자의 기기에만 저장되며, 앱을 삭제하면 모든 데이터가 함께 삭제됩니다.

5. 아동의 개인정보

본 앱은 만 14세 미만의 아동을 대상으로 하지 않으며, 의도적으로 아동의 개인정보를 수집하지 않습니다.

6. 개인정보처리방침의 변경

본 방침이 변경되는 경우, 앱 업데이트 또는 이 페이지를 통해 공지합니다.
